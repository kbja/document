# 																				List

## 一、ArrayList

### 1、基础知识

​		基于数组实现，允许插入null值。随机访问get/写入set花费O(1)，对连续插入不友好(需进行扩容)。和**Vector**大致相同，但是不支持并发访问。

​		连续的容器，必须前面有值才能向后继续插入。

​		add(int index,E ele)是在index位置插入一个元素，并将index之后的元素向后移动一位。

### 2、重点

* 当需求容量>当前数组容量时，会进行扩容，扩容大小为之前大小的**1.5倍**。**通过右移实现**
* 有些JVM会保留数组的头部空间，所以最大的数组大小为Integer.MAX-8，以防止OOM。
* 删除元素时，应使用iterator，使用循环的话，应从后向前



## 二、LinkedList

### 1、基础知识

​		基于**List和Deque**接口实现的双向链表，允许插入null值，对连续插入友好，对随机访问不友好(花费O(n))

​		可以实现**栈、双向链表、双头队列**的功能

 * 栈 push() pop()

 * queue 

   | --     | add()    | offer()  | remove()       | poll()         | elemet()  | peek()   |
   | :----- | -------- | -------- | -------------- | -------------- | --------- | -------- |
   | 功能   | 头部插入 | 头部插入 | 头部检索且删除 | 头部检索且删除 | 头部检索  | 头部检索 |
   | 抛异常 | 容量限制 |          | queue为空      |                | queue为空 |          |

### 2、重点

* 删除元素时，应使用iterator，使用循环的话，应从后向前
